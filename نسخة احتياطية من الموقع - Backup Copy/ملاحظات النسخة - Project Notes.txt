🗄️ ملخص جداول قاعدة البيانات - Database Tables Summary:
========================================================

1. 👥 users - جدول المستخدمين
   - معلومات المستخدمين المسجلين

2. 🛠️ services - جدول الخدمات الأساسية
   - الخدمات الرئيسية (تصميم، تسويق، مونتاج، إلخ)

3. 🔧 sub_services - جدول الخدمات الفرعية
   - الخدمات التفصيلية تحت كل خدمة أساسية

4. 📦 packages - جدول الباقات
   - باقات الخدمات المختلفة

5. 🛒 orders - جدول الطلبات
   - طلبات العملاء

6. 📋 order_items - جدول عناصر الطلبات
   - تفاصيل كل طلب

7. 💳 subscriptions - جدول الاشتراكات
   - اشتراكات العملاء في الباقات

8. 🧾 receipts - جدول الإيصالات
   - إيصالات الدفع المرفوعة

9. 🏠 properties - جدول العقارات
   - بيانات العقارات للبيع والشراء

10. 👨‍💼 admins - جدول المديرين
    - حسابات المديرين وصلاحياتهم

11. 🎨 portfolio - جدول سابقة الأعمال
    - أعمال الشركة السابقة

12. ⚙️ system_settings - جدول إعدادات النظام
    - إعدادات الموقع العامة

13. 🔗 additional_links - جدول الروابط الإضافية
    - روابط التواصل الاجتماعي الإضافية

14. 💰 payment_methods - جدول طرق الدفع
    - طرق الدفع المتاحة

📝 ملاحظات مهمة - Important Notes:
==================================

1. تأكد من إعداد متغيرات البيئة (.env.local)
2. قم بإنشاء مشروع Supabase جديد
3. استورد هيكل قاعدة البيانات من مجلد "هيكل قاعدة البيانات"
4. قم بإعداد Storage buckets في Supabase
5. تأكد من تفعيل RLS policies حسب الحاجة

🚀 للبدء - Getting Started:
===========================

1. npm install
2. إعداد ملف .env.local
3. استيراد قاعدة البيانات
4. npm run dev
