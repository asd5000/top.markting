# توب ماركتنج - دليل الإعداد

## نظرة عامة
نظام متكامل لإدارة خدمات التسويق الرقمي والوساطة العقارية مع لوحة تحكم إدارية شاملة.

## المتطلبات
- Node.js 18+ 
- npm أو yarn
- حساب Supabase

## خطوات الإعداد

### 1. إعداد Supabase

1. **إنشاء مشروع جديد:**
   - اذهب إلى [supabase.com](https://supabase.com)
   - أنشئ حساب جديد أو سجل دخول
   - أنشئ مشروع جديد

2. **الحصول على بيانات الاتصال:**
   - اذهب إلى Settings > API
   - انسخ `Project URL` و `anon public key` و `service_role key`

3. **إعداد قاعدة البيانات:**
   - اذهب إلى SQL Editor في لوحة تحكم Supabase
   - انسخ محتوى ملف `supabase/schema.sql` والصقه
   - اضغط Run لتنفيذ الاستعلام

### 2. إعداد المتغيرات البيئية

1. **إنشاء ملف البيئة:**
   ```bash
   cp .env.local.example .env.local
   ```

2. **تحديث المتغيرات:**
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
   ```

### 3. تثبيت التبعيات وتشغيل المشروع

```bash
# تثبيت التبعيات
npm install

# إعداد قاعدة البيانات
node scripts/setup-supabase.js

# تشغيل المشروع
npm run dev
```
