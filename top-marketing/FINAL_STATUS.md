## 🔧 التقنيات المستخدمة

- **Backend**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth (جاهز للتفعيل)

- **Styling**: RTL Support + Arabic Fonts (Cairo, Tajawal)
- **State Management**: React Hooks
- **Real-time**: Supabase Real-time subscriptions
